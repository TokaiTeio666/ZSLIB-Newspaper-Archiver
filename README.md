# ZSLIB-Newspaper-Archiver(基于 python-docx 与 Selenium)

## 项目简介
本项目是一个自动化工具，旨在从数字图书馆平台批量采集历史报纸资源。它能够自动处理复杂的网页翻页逻辑，精确裁剪高清原图，并将每一篇报纸条目按照标准化的格式保存为独立的 Word (.docx) 文档，便于后续的史料研究与整理。

## 核心功能
- **自动化采集**：支持多页结果自动翻页，直至尾页。
- **高清原图裁剪**：通过 `PIL (Pillow)` 库计算原图与网页显示的比例，实现坐标精准对齐的高清裁剪。
- **标准化文档生成**：自动解析报刊名、日期及版次信息，生成符合排版要求的 Word 文档。
- **智能文件命名**：自动清洗非法字符，以“日期_报刊名”命名，确保文件系统兼容性。

## 技术栈
- **Python 3.x**
- **Selenium**: 网页自动化与动态内容交互。
- **python-docx**: 生成并操作 Word 文档。
- **Pillow (PIL)**: 图像处理与高清裁剪坐标计算。
- **Requests**: 负责高清大图的二进制数据下载。
